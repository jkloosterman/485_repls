# EECS 485 Kloosterman repl.it Exercises
repl.it decided to shut down public projects, so the lecture exercises that used to be hosted on repl.it are now posted here.

To use these, clone the repository to your computer:
```
git clone git@github.com:jkloosterman/485_repls.git
```

The exercises are in each folder. The `run.sh` script is a way to execute the exercise, similar to pressing the play button on repl.it.
